## Usage

### Run Locally

1. Clone the repository:

```bash
git clone https://github.com/Anne-Okingo/Contact-Form.git
```
```bash
cd contact-form
```
2. Install dependencies:

```bash
npm install
```
3. Start the development server:

```bash
npm start
```
